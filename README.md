# Knight's Puzzle

A cozy, account-free knight's-tour puzzle game. Each level is a partial
checkerboard with a **start** and **end** square; you win by hopping the knight
so it visits every available square exactly once and finishes on the end square.

Puzzles are generated by walking a knight randomly (no revisits) across an
N×N board — the visited squares become the level, everything else is removed.
Because the puzzle is built from a real knight's walk, it is **always solvable
by construction** (no solver needed).

## Stack

Vite + React + TypeScript, pure client-side. No server, no accounts, no
database.

## Develop

```sh
bun install
bun run dev        # local dev server (Vite)
```

## Gates

```sh
bun run typecheck  # tsc, no emit
bun run lint       # eslint
bun run test       # bun unit tests (engine)
bun run build      # vite production build
bun run smoke      # headless Chrome smoke (Playwright) — reserved port 4317
bun run ci         # format:check + lint + typecheck + test + build
```
